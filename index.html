⚙ Подключить свой ИИ — качество «как в чате»

Базовый режим работает без всего — просто загрузите файл: контент переносится с фирменным оформлением. Чтобы получить полноценную переверстку (плитки, колонки, «картинка + текст») — подключите свой ИИ одним из двух способов.

Способ 1 — через своего ИИ-агента безопасно, без ключей

Скачайте инструкцию и вставьте её в свой Claude / ChatGPT / Cursor (ChatGPT → Custom Instructions; Claude → Settings → Capabilities → Skills → Add), затем приложите презентацию — агент переверстает её в вашей среде. Ни ключи, ни файлы никуда не уходят.

Способ 2 — вставить свой API-ключ (для продвинутых)

Сайт вызовет вашу модель напрямую из браузера. Из браузера работают ключи Anthropic (sk-ant-…) и OpenRouter (sk-or-…) — провайдер определяется по ключу автоматически.

ID модели (по умолчанию подставится)

Безопасность. Ключ хранится только в этой вкладке и стирается при её закрытии; уходит напрямую к провайдеру — на наши серверы он не попадает (их нет). Совет: заведите ключ с ограниченным лимитом расходов, чтобы даже при утечке нельзя было «сжечь» больше заданного. Максимальная безопасность — Способ 1 (вообще без ключа).
